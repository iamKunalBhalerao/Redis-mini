Why Redis

In-memory speed

Reads and writes at sub-millisecond latency — ideal for caches, flags, and sessions.

Key/value simplicity

Store banners, tokens, or config with a single SET. Retrieve instantly with GET.

Health visibility

PING lets services report Redis availability alongside their primary database.

Getting started

01 / prerequisites

Node.js 18+, pnpm, Redis on localhost:6379, optional MongoDB.

02 / install

Install all workspace dependencies in one command.

pnpm install
03 / run a service

Use Turborepo filters to run any single app in isolation.

pnpm dev --filter site-banner
04 / build

Compile all apps for production in a single pass.

pnpm build
